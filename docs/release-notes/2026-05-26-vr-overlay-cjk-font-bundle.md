# VR Subtitle Overlay CJK Font Bundle Release Note

- The Windows installer size intentionally increases by about 18 MB because releases now bundle the official `NotoSansCJK-Medium.ttc` font for deterministic Korean, Japanese, Simplified Chinese, and Traditional Chinese VR subtitle rendering.
- The bundled Noto Sans CJK Medium TTC is staged at `puripuly_heart/data/fonts/NotoSansCJK-Medium.ttc` in the packaged app tree and carries SIL Open Font License 1.1 provenance under `third_party/noto-sans-cjk/`.
